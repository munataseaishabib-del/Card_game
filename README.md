游戏流程
路线图界面：点击可选择的节点（红色为战斗、蓝色为商店、紫色为事件）
战斗界面：
点击卡牌使用（需要足够能量）
点击 "结束回合" 让敌人行动
奖励界面：选择一张卡牌添加到牌组，或点击跳过
商店界面：用金币购买卡牌
事件界面：阅读剧情并选择选项
游戏结束：生命归零或到达第 10 层
